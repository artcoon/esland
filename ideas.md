## Idea 1: Neomorphic Tech-Chamber (네오모피즘 테크-챔버)

### 1. Design Movement
**Neomorphism & Glassmorphism Fusion** - 고도로 연마된 금속, 유리, 그리고 입체적인 깊이감을 주는 그림자를 활용하여 첨단 엔지니어링과 프리미엄 컨설팅의 조화를 표현합니다.

### 2. Core Principles
* **Tactile Depth**: 모든 카드와 다이어그램 요소가 만질 수 있을 것 같은 입체감을 가집니다.
* **Reflective Surfaces**: 샴페인 골드와 딥 퍼플의 은은한 그라데이션이 유리를 통과하는 듯한 굴절 효과를 구현합니다.
* **Precision Layout**: 중앙 정렬을 피하고, 섀도우와 보더의 정교한 픽셀 매칭을 통해 공학적인 신뢰감을 줍니다.

### 3. Color Philosophy
* **Background**: #0F0E17 (Graphite Dark - 심해의 어둠처럼 차분하고 묵직한 배경)
* **Accent 1**: #A78BFA (Deep Purple - 혁신과 고귀함)
* **Accent 2**: #F59E0B (Champagne Gold - 성공과 프리미엄 자산)
* **Surface**: #1F1D2E (약간의 자색이 감도는 다크 그레이로 고급스러운 레이어 구성)

### 4. Layout Paradigm
**Asymmetric Dashboard Grid** - 대칭적인 레이아웃 대신, 좌우 여백을 다르게 배치하고 비정형적인 카드 그리드를 활용하여 데이터 중심의 첨단 컨설팅 허브 느낌을 줍니다.

### 5. Signature Elements
* **Glassmorphic Interactive Blueprint**: 다층복합보강구조 시스템을 입체적인 유리 레이어로 구현하여 마우스 오버 시 각 층이 분리되어 솟아오르는 효과.
* **Gold-accented Micro-shadows**: 버튼과 중요 카드에 샴페인 골드 테두리와 아주 미세한 네온 글로우 효과 적용.

### 6. Interaction Philosophy
* 모든 마우스 호버는 '누름(Depressed)' 또는 '띄움(Elevated)'의 물리적 반응을 동반합니다.
* 마우스 커서의 움직임에 반응하여 배경의 미세한 파티클이나 그리드가 부드럽게 일렁입니다.

### 7. Animation Guidelines
* **Entrance**: 0.3초 이내의 부드러운 스프링 물리 엔진 느낌의 스케일업 (`cubic-bezier(0.34, 1.56, 0.64, 1)`)
* **Transition**: 카테고리 전환 시 카드들이 흩어졌다가 모이는 모핑 애니메이션.

### 8. Typography System
* **Display/Heading**: Playfair Display (세련되고 클래식한 Serif로 신뢰감과 럭셔리함 부여)
* **Body**: Pretendard (가독성이 극대화된 현대적 Sans-serif)

## Idea 2: Minimalist Royal Editorial (미니멀리스트 로열 에디토리얼)

### 1. Design Movement
**Premium Editorialism (고급 잡지/스위스 타이포그래피 스타일)** - 극도의 절제미와 압도적인 타이포그래피 크기 대비를 사용하여 명품 브랜드의 카탈로그 같은 우아함을 자아냅니다.

### 2. Core Principles
* **Extreme Contrast**: 매우 큰 헤드라인과 아주 작고 정교한 메타 데이터의 극단적 대비.
* **Negative Space as Canvas**: 공백을 여백이 아닌 '숨구멍'이자 '집중의 도구'로 활용.
* **Organic Flow**: 스크롤에 따라 텍스트와 이미지가 우아하게 미끄러지듯 등장하는 시네마틱 흐름.

### 3. Color Philosophy
* **Background**: #FAF6F0 (Warm Champagne - 부드럽고 따뜻한 상아색, 샴페인 골드의 무광택 버전)
* **Text/Primary**: #1A1A24 (Deep Graphite - 검은색에 가까운 진한 그래파이트로 잉크 같은 질감)
* **Accent**: #581C87 (Royal Deep Purple - 깊고 우아한 퍼플을 포인트 컬러로 사용)

### 4. Layout Paradigm
**Single Column Narrative** - 복잡한 다단 그리드를 배제하고, 하나의 거대한 이야기 흐름을 따라 스크롤하며 몰입하는 스토리텔링 레이아웃.

### 5. Signature Elements
* **Monolithic Image Reveal**: 스크롤 시 대표이사의 고화질 세로형 인물 사진이 부드럽게 페이드인되며 배경과 경계 없이 녹아드는 효과.
* **Serif Monogram**: 'S'와 'C'가 결합된 극도로 단순하고 우아한 골드 모노그램 로고.

### 6. Interaction Philosophy
* 마우스 호버 시 텍스트 아래에 퍼플/골드 라인이 왼쪽에서 오른쪽으로 흐르듯 그려지는 정교한 드로잉 효과.
* 링크 클릭 시 화면 전체가 우아하게 암전되었다가 켜지는 시네마틱 트랜지션.

### 7. Animation Guidelines
* **Scroll-triggered Fade**: 스크롤 위치에 따라 요소들이 부드럽게 솟아오르며 등장 (`cubic-bezier(0.25, 1, 0.5, 1)`)
* **Duration**: 0.6초~0.8초의 비교적 여유롭고 우아한 타이밍 적용.

### 8. Typography System
* **Display/Heading**: Cormorant Garamond (극도로 얇고 우아한 세리프체)
* **Body**: Lora (서정적이고 품격 있는 본문용 세리프) + Pretendard (메타데이터용)

## Idea 3: Liquid Carbon & Gold (리퀴드 카본 & 골드) - ★ 최종 선택 ★

### 1. Design Movement
**Luxury Dark Mode & Generative Fluid Art** - 딥 퍼플과 그래파이트의 어두운 우주 공간 속에 샴페인 골드빛 유체(Liquid)와 입자가 흐르는 듯한 럭셔리 디지털 아트 감성. 현대 엔터프라이즈 리더들의 이성을 자극하는 데이터 시각화와 감성을 자극하는 유체 모션의 결합.

### 2. Core Principles
* **Luxury Dark Palette**: 어두운 그래파이트와 딥 퍼플의 조합에 빛나는 샴페인 골드를 매치하여 압도적인 프리미엄 감성 전달.
* **Fluid Interactive Diagrams**: 정적인 다이어그램 대신 마우스의 움직임이나 스크롤에 반응하는 역동적 인터랙션 요소 도입.
* **Enterprise Credibility**: 재무 지표, 시공 실적, 특허 구조를 정교한 대시보드 UI로 시각화하여 컨설팅의 전문성을 입증.

### 3. Color Philosophy
* **Primary (Deep Purple)**: #2E1065 / #1E1B4B (깊은 자색과 남색의 경계, 심오한 전문성과 고급스러움)
* **Accent (Champagne Gold)**: #FBBF24 / #D97706 (눈부시게 빛나는 샴페인 골드, 가치와 혁신)
* **Base (Graphite Dark)**: #0B0F19 / #111827 (완전한 블랙이 아닌, 블루/퍼플 기운이 도는 세련된 다크 그래파이트)
* **Muted**: #9CA3AF (가독성을 확보하는 은은한 실버 그레이)

### 4. Layout Paradigm
**Interactive Dashboard & Cinematic Split**
* 영웅 섹션(Hero)은 대표이사 사진과 웅장한 브랜드 슬로건이 대비를 이루는 Cinematic Split 레이아웃 적용.
* 기술 및 실적 소개는 세련된 대시보드 형태로 구성하여 실제 데이터를 탐색하는 듯한 지적 흥미 유발.

### 5. Signature Elements
* **Interactive 5-Layer Slope Diagram**: 다층복합보강구조 시스템의 5개 층을 입체적이고 역동적인 아코디언/탭 형태로 시각화하여 각 층의 역할(식생, 바이오차, 지오그리드 등)을 마우스 인터랙션으로 확인.
* **KPI Before/After Dashboard**: 산사태 복구 및 탄소중립 시공의 놀라운 전/후 지표를 슬라이더 및 인터랙션 차트로 구현.
* **Multi-step Qualification Funnel**: 리드가 자신의 기업 규모와 필요 서비스를 입력하면 즉시 맞춤형 진단을 제안하는 인터랙션 폼.

### 6. Interaction Philosophy
* 모든 카드와 버튼은 호버 시 미세한 골드 글로우 테두리가 켜지며 부드럽게 솟아오름 (`transform: translateY(-4px)`).
* 대시보드의 숫자(KPI)는 페이지 스크롤 시 0부터 부드럽게 올라가는 카운트업 애니메이션 적용.

### 7. Animation Guidelines
* **Ease**: `--ease-out: cubic-bezier(0.16, 1, 0.3, 1);` (초기 속도가 매우 빠르고 끝이 극도로 부드러운 감속 곡선)
* **Micro-interactions**: 버튼 활성화 시 골드 파티클이 사방으로 퍼지는 듯한 쾌적한 피드백 제공.

### 8. Typography System
* **Display/Heading**: Cinzel / Playfair Display (웅장하고 권위 있는 프리미엄 세리프)
* **Body**: Pretendard (엔터프라이즈 비즈니스에 최적화된 고해상도 Sans-serif)
